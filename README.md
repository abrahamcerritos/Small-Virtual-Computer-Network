# Small Virtual Office Network Design and Simulation

## Project Overview

This project demonstrates the design and testing of a small virtual office network using Cisco Packet Tracer. The network includes one router, one Ethernet switch, three client computers, and one server.

## Network Devices

- 1 router
- 1 Ethernet switch
- 3 computers: PC0, PC1, and PC2
- 1 server

## Technologies and Protocols

- Cisco Packet Tracer
- Ethernet
- IPv4
- IPv6
- ARP
- ICMP
- ICMPv6
- DHCP
- DNS

## Project Features

- IPv4 and IPv6 addressing
- Default gateway configuration
- Communication between client computers
- IPv4 and IPv6 ping testing
- Network traffic testing and analysis

## Testing

The network was tested using ping commands between PC0, PC1, and PC2. The successful results confirmed that the devices were connected and communicating correctly.
